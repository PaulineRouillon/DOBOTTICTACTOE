Lancer le programme :
	en ligne de commande :	py app.py
	executer le fichier Run_App.bat

Réinitialiser le robot :
	Lancer l'interface DobotStudio, 
	Se connecter au robot, 
	Cliquer sur "HOME" (en haut a droite de l'interface)
	Se déconnecter du robot

Si problème de communication avec le robot: (port busy)
	aller dans le gestionnaire de périphérique
	réinitialiser les pilotes
